# Infoxel (*2018 - 2020*)

**Full-Stack Software Engineer**

Worked on internal dashboards and led the migration of a 15-year-old monolith into cloud-native, distributed microservices. Designed a pipes-and-filters architecture for large-scale media ingestion, processing millions of audio fingerprints daily. Drove modernisation efforts across backend, frontend and infrastructure while improving reliability, performance and delivery speed.


## Highlights
- Designed Django REST APIs enabling communication across new microservices.
- Integrated heterogeneous data stores and built monitoring dashboards with Grafana.
- Improved backend performance through query optimisation and strategic caching.
- Architected and implemented the full media-processing pipeline, including extraction and matching of audio fingerprints.
- Drove the effort to implement an ETL ingesting and reconciling program metadata.
